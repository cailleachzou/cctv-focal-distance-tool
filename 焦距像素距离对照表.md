# 焦距 × 像素 × 距离对照表

## 用途

安防摄像机选型时，**根据场景需要的"看清程度"反推焦距和像素**。
核心思路：先定 DORI 等级 → 反查距离 → 选焦距和像素。

## 计算参数

| 参数 | 取值 | 来源 |
|---|---|---|
| 传感器 | 1/2.8"（Sensor_W = 5.12mm） | Hikvision / Dahua 主流 IP 摄像机 |
| 焦距 | 2.8 / 4 / 6 / 8 / 12 mm | 常见定焦镜头 |
| 像素 | 2MP (1920H) / 4MP (2560H) / 8MP (3840H) | 1080P / 2K / 4K |

**DORI 标准**（IEC 62676-4）：

| 等级 | 英文 | 含义 | 像素密度 (px/m) |
|---|---|---|---|
| **D** 探测 | Detection | 能"看到有东西" | 25 |
| **O** 观察 | Observation | 能看清轮廓/姿态 | 62 |
| **R** 识别 | Recognition | 能认出熟人 | 125 |
| **I** 辨识 | Identification | 能辨认陌生人五官 | 250 |

**核心公式**：
- 水平视场角 `H-FOV = 2 × arctan(5.12 / (2 × f))`
- 垂直视场角 `V-FOV = 2 × arctan(3.84 / (2 × f))`  ← 1/2.8" 传感器垂直方向
- DORI 距离 `D = H_Pixels / (2 × tan(H-FOV/2) × DORI_px/m)`
- 盲区 `B = H / tan(α + V-FOV/2)`  ← H=2.8m 安装高度，α 为俯角

**盲区假设**（不同焦距典型安装俯角）：

| 焦距 | 典型目标距离 | 俯角 α | 计算说明 |
|---|---|---|---|
| 2.8mm | 5m | 29.2° | 广角镜头看脚下到 5m 范围 |
| 4mm | 8m | 19.3° | 标准走道看 5-10m |
| 6mm | 15m | 10.6° | 中距覆盖 10-20m |
| 8mm | 20m | 8.0° | 远距覆盖 15-25m |
| 12mm | 30m | 5.3° | 长焦看 25-40m |

> 实际项目可调俯角：增大俯角 → 盲区缩小但远距覆盖减少；减小俯角 → 反之。

## 速查指南

### 怎么用这张表

1. **先定 DORI 等级** — 客户需要"看到有人"还是"看清脸"？
   - 一般入侵检测 → D 级（25 px/m）
   - 行为分析 / 跌倒检测 → O 级（62 px/m）
   - 门禁考勤 / VIP 识别 → R 级（125 px/m）
   - 公安取证 / 反查 → I 级（250 px/m）

2. **再定距离** — 目标在多少米内需要满足？

3. **查表选焦距和像素** — 选最经济的组合（不堆配置）

### 经验法则

- **像素翻倍 ≠ 距离翻倍**：4MP 相对 2MP 的 DORI 距离仅提升 33%，但存储和带宽翻倍
- **焦距翻倍 → 视场宽减半**：6mm 改 12mm 视场宽从 8.5m 降到 4.3m，需要更多点位覆盖
- **同距离下，焦距+像素可互换**：12m 距离要 R 级，4mm@8MP（24m 裕量）和 8mm@4MP（32m 裕量）都达标，但前者广角更适合大场景
- **焦距越长，盲区越大**：12mm 镜头 2.8m 安装时盲区 10.9m，路边停车 / 摄像头正下方是盲区；需要补点或换广角

### 盲区注意事项

- **盲区内不布防**：相机正下方的扇形区域（半径 = 盲区值）是"看不见的死角"，入侵检测、跌倒检测在此区域无效
- **电梯厅 / 前台**：用 2.8mm 广角，盲区仅 1.4m，几乎全覆盖
- **停车场立杆**：用 6-8mm 镜头，盲区 5-7m，立杆周围 5m 内要靠邻杆覆盖
- **周界立杆**：相邻两杆的盲区不能重叠太大（否则中间漏防），一般盲区占立杆间距的 15-20% 为宜

## 备注

- 本表为**理论计算值**（理想光学 + 标准 DORI），实际项目应留 30-50% 裕量（雾天、逆光、镜头畸变等会降低有效距离）
- 镜头标注的焦距可能为"等效焦距"（厂商虚标），需以实测视场角为准
- 变焦镜头可覆盖多档，但成本是定焦 3-5 倍，选型时优先定焦
- 1/3" 传感器的视场角比 1/2.8" 小约 6%，DORI 距离相应降低 6%
- 数据来源：IEC 62676-4 DORI 标准 + 主流厂商镜头参数（海康、大华、宇视）
